# Pesticide-Pass
2022学期1 Android应用大作业

**v1.0 已经完成**

## 功能

- 从样品图片中提取亮度，拟合得到 亮度-农药浓度 线性模型。

- 从保存在本地的模型进行预测，得到样品的农药浓度预测值。

- 管理本地和云端的模型列表。
